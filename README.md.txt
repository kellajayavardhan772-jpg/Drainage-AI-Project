# Smart Village Drainage AI Model

## Project Overview
This project uses Artificial Intelligence, Machine Learning, and Deep Learning to detect drainage-related health risks in villages.

## Problem Statement
In many villages, drainage systems do not have proper exit channels. Due to this:
- Water gets stagnant
- Plastic and garbage block the flow
- Mosquito breeding increases
- Diseases like malaria and dengue may spread
- During rainy season, overflow increases

## Objective
To build an AI-based system that can:
- Detect blocked drainage
- Detect clear drainage
- Predict health risk zones
- Help suggest proper drainage exit management

## Technologies Used
- Python
- TensorFlow / Keras
- CNN (Convolutional Neural Network)
- Google Colab / VS Code

## Dataset
The dataset contains two classes:
- blocked
- clear

## Files
- `train_model.py` → Trains the AI model
- `predict.py` → Predicts whether drainage is blocked or clear

## Output
The system predicts:
- Blocked Drainage
- Clear Drainage
